# Privacy

`YouTube Floating PiP` does not collect, store, transmit, or sell personal data.

## What the extension accesses

- The active tab you explicitly invoke it on
- The current page's existing `<video>` element on supported YouTube pages

## What the extension does not do

- No analytics
- No tracking
- No external API calls
- No remote code
- No account system
- No background syncing
- No data storage

## Permissions

- `activeTab`
  Used only after you click the extension or trigger its shortcut, so the extension can interact with the current YouTube tab.
- `scripting`
  Used to run a small script in the current tab that asks Chrome to open or close the native Picture-in-Picture window.

## Summary

This extension is intentionally local-first and minimal. It exists to do one thing: open the current YouTube video in Chrome's detached Picture-in-Picture window.
